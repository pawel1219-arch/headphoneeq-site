FAQ

Is it free?

HeadphoneEQ is free to download. You can try any preset for 30 seconds. A one-time Pro upgrade ($4.99) unlocks unlimited use, custom EQ, favorites, and auto-gain modes.

Does it work with Spotify / Apple Music / YouTube?

Yes. HeadphoneEQ captures system audio, so it works with every app on your Mac.

What macOS versions are supported?

macOS 13 (Ventura) and later. System Audio Capture (recommended) requires macOS 14.2+. Older versions can use BlackHole virtual audio driver.

Does it add latency?

Minimal. Processing happens in real time with optimized 64-bit biquad filters. Typical latency is under 10ms.

What is AutoEQ?

AutoEQ is an open-source project that generates headphone equalization corrections based on scientific measurements. HeadphoneEQ uses these corrections to make your headphones sound closer to the Harman target curve — the frequency response preferred by most listeners.
